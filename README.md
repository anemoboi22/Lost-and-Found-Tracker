# Lost-and-Found-Tracker

Lost and Found Tracker was created of mine as a project in iOS Mobile Application Cross Platform - Xamarin Forms. It was designed as a tool for my fellow schoolmates to report and post their lost belongings. Lost and Found Tracker was designed as simple, quick, convenient, and modernly aesthetic. The languages used are Xamarin C# and SQLite.
